변수
ㄴ 첫글자 소문자, 자료형 형식은 뒤에 붙이기
EX) playerHp, playerTr

함수

ㄴ 첫글자 대문자, 함수명은 동사-명사순으로
Ex) CalcPrice, AlignmentInventory

구조체, 열거형, static변수들, const 변수들은 따로 스크립트 빼기
